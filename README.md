# Wordpress Android Client
An easy android client for wordpress blog~ 

Click [here: Wordpress Api](http://github.com/httpbin/wordpress-api) to get the server client code (The wordpress plugin for creating apps).
 
## Functions list:
1. Posts
 * List
 * Details
2. Pages
 * List
 * Details
3. Categories
 * List
4. Blog searching
5. Friend links
6. Sliding images
7. Starting image
